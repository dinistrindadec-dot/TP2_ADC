main module¶

Aplicação de gestão de biblioteca (catálogo e requisições)¶

Programa principal com menus interativos para o administrador (criar livros, ver requisições) e para leitores (consultar catálogo, requisitar e devolver).

Os módulos modulo_catalogo, modulo_requisicoes, modulo_utilizadores e modulo_backup concentram a lógica de negócio e a persistência em ficheiros JSON.

Autor:

TP2 ADC

Versão:

1.0

main.fluxo_admin_backup()[código fonte]¶

Fluxo do administrador: criação de backup dos ficheiros de dados.

Returns:

None

main.fluxo_admin_catalogo()[código fonte]¶

Fluxo do administrador: listagem do catálogo completo.

Returns:

None

main.fluxo_admin_criar_livro()[código fonte]¶

Fluxo do administrador: registo de um novo livro no catálogo.

Solicita título/autor (obrigatórios), número de exemplares (>= 1) e um tema opcional para permitir pesquisa.

Returns:

None

main.fluxo_admin_fichas()[código fonte]¶

Fluxo do administrador: consulta da ficha de um utilizador.

Mostra o índice de utilizadores e pede o id para apresentar a ficha detalhada (incluindo resumo de requisições).

Returns:

None

main.fluxo_admin_registar_utilizador()[código fonte]¶

Fluxo do administrador: registo de um novo utilizador.

O nome é obrigatório. E-mail e telefone são opcionais.

Returns:

None

main.fluxo_admin_stock()[código fonte]¶

Fluxo do administrador: gestão de stock (nº exemplares por livro).

Este fluxo consulta o stock atual e permite alterar o número total de exemplares de um livro. Antes de alterar, valida que o novo stock não fica abaixo do número de exemplares atualmente requisitados.

Returns:

None

main.fluxo_admin_ver_requisitados()[código fonte]¶

Fluxo do administrador: listagem de requisições ativas.

Returns:

None

main.fluxo_leitor_catalogo()[código fonte]¶

Fluxo do leitor: consulta do catálogo com disponibilidade.

Returns:

None

main.fluxo_leitor_contactos()[código fonte]¶

Fluxo do leitor: atualização de contactos (e-mail e telemóvel).

Returns:

None

main.fluxo_leitor_devolver()[código fonte]¶

Fluxo do leitor: devolução de um livro (encerra uma requisição).

Mostra as requisições ativas do utilizador e pede o #id da requisição para efetuar a devolução.

Returns:

None

main.fluxo_leitor_pesquisar()[código fonte]¶

Fluxo do leitor: pesquisa de livros por autor ou tema.

A pesquisa é feita no catálogo e, em seguida, os resultados são formatados com disponibilidade (usando as requisições ativas).

Returns:

None

main.fluxo_leitor_prazos()[código fonte]¶

Fluxo do leitor: consulta dos prazos das requisições ativas.

Returns:

None

main.fluxo_leitor_requisitar()[código fonte]¶

Fluxo do leitor: criação de uma requisição de livro.

O leitor escolhe o seu id de utilizador e depois seleciona o livro a requisitar. A criação e validações são delegadas a modulo_requisicoes.

Returns:

None
