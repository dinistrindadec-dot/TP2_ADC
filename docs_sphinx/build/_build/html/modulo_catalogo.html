modulo_catalogo module¶

Gestão do catálogo de livros (persistência em JSON).

Este módulo expõe operações para o administrador manter o catálogo atualizado e para qualquer perfil listar os livros disponíveis para consulta.

modulo_catalogo.adicionar_livro(titulo, autor, exemplares, tema='')[código fonte]¶

Regista um novo livro no catálogo.

Args:

titulo (str): Título do livro. autor (str): Nome do autor. exemplares (int): Número de exemplares físicos (>= 1). tema (str): Tema ou género (opcional), usado na pesquisa.

Returns:

dict: O dicionário do livro criado e persistido.

modulo_catalogo.atualizar_exemplares(livro_id, novo_total)[código fonte]¶

Atualiza o número total de exemplares de um livro.

Nota: esta função não valida requisições ativas. Essa validação é feita no fluxo do main.py antes de chamar este método (para garantir que o stock não fica abaixo do número de exemplares atualmente emprestados).

Args:

livro_id (int): Identificador do livro. novo_total (int): Novo stock total (>= 1).

Returns:

tuple[bool, str]: Resultado da operação e mensagem para o utilizador.

modulo_catalogo.carregar_livros()[código fonte]¶

Carrega todos os livros do ficheiro de dados.

Returns:
list[dict]: Lista de livros, onde cada livro contém:

  • id (int)

  • titulo (str)

  • autor (str)

  • exemplares (int)

  • tema (str, opcional)

modulo_catalogo.listar_catalogo_formatado()[código fonte]¶

Devolve linhas de texto com o catálogo completo para apresentação na

consola.

Returns:

str: Texto com um livro por linha (id, título, autor, exemplares).

modulo_catalogo.obter_livro(livro_id)[código fonte]¶

Obtém um livro pelo identificador.

Args:

livro_id (int): Identificador do livro.

Returns:

dict | None: Dicionário do livro, ou None se não existir.

modulo_catalogo.pesquisar_por_autor(termo)[código fonte]¶

Pesquisa livros por autor (correspondência parcial, case-insensitive).

Args:

termo (str): Texto (parcial) a procurar no campo autor.

Returns:

list[dict]: Lista de livros cujo autor contém o termo.

modulo_catalogo.pesquisar_por_tema(termo)[código fonte]¶

Pesquisa livros por tema/género (correspondência parcial).

Args:

termo (str): Texto (parcial) a procurar no campo tema.

Returns:

list[dict]: Lista de livros cujo tema contém o termo.
