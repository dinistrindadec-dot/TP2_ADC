modulo_backup module¶

Cópias de segurança (backup) dos ficheiros de dados.

Este módulo cria uma cópia de segurança dos ficheiros JSON usados pela aplicação (catálogo, requisições e utilizadores).

A estratégia é simples: - Cria (se necessário) a pasta data/. - Cria (se necessário) a pasta data/backups/. - Para cada ficheiro esperado, se existir, copia-o para uma nova subpasta

backup_<timestamp>/.

  • Se nenhum ficheiro existir, remove a pasta de backup criada e devolve (None, []).

modulo_backup.criar_backup()[código fonte]¶

Cria um backup dos ficheiros de dados existentes.

Returns:
tuple[Path | None, list[str]]:

  • Caminho para a pasta de backup criada (ou None se não foi copiado nenhum ficheiro).

  • Lista de nomes de ficheiros copiados.

Raises:

OSError: Se houver falhas de IO ao criar pastas ou copiar ficheiros.
